Curso de Tailwind CSS

O Tailwind CSS é um framework de utilitários CSS.
Diferente do Bootstrap (que já vem com componentes
prontos como botões, cards e navbars), o Tailwind
te dá classes utilitárias que você combina para criar
seu próprio design.
⚡ Vantagens:
⚡Altamente personalizável → Você monta o design do jeito que quiser.
⚡Produtividade → Sem precisar criar CSS do zero, basta usar as classes.
⚡Responsivo fácil → Classes como sm:, md:, lg: ajustam o layout em diferentes telas.
⚡Performance → Remove CSS não usado na build final, deixando o site leve.
