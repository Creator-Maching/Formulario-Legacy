Curso de Chakra IU

O Chakra UI é uma biblioteca de componentes para React
que facilita a criação de interfaces modernas e acessíveis.
Ele fornece componentes prontos, mas com muita flexibilidade,
permitindo que você personalize estilos usando props no próprio
JSX sem precisar escrever CSS separado.
⚡ Vantagens do Chakra UI:
⚡Componentes acessíveis – já seguem boas práticas de acessibilidade (teclado, leitores de tela).
⚡Estilo via props – você pode definir cores, margens, espaçamentos e tamanhos direto no JSX.
⚡Tema centralizado – define cores, fontes e estilos globais para toda a aplicação.
⚡Responsividade fácil – com props como base, md, lg você ajusta o layout em diferentes telas.
⚡Integração com React – feito para funcionar naturalmente com React, incluindo hooks e state.
